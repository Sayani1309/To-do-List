# To-Do List Web App

A simple and functional To-Do List application built using
HTML, CSS, and Vanilla JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using localStorage

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in your browser

## Future Improvements
- Edit tasks
- Due dates
- Dark mode
